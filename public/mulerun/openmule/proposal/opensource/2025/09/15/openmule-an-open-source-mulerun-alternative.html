OpenMule: An Open Source MuleRun Alternative

Sep 15, 2025 • James Brown

Mulerun is a great marketplace platform hosting online CUA agents as well as traditional workflow AI applications. However, it is not open source, which means that we cannot contribute to it directly. Therefore, we propose to create an open source alternative to Mulerun, which we call OpenMule.

OpenMule is an open source marketplace platform that allows users to create and deploy CUA agents and traditional workflow AI applications without the need for a third-party marketplace. It is designed to be easy to use and and easy to develop on.

It differs from Mulerun in the following ways:

  1. Open Source: OpenMule is open source, which means that it is free to use and modify. This allows users to contribute to the project and improve it over time.
  2. Relay as a Service: OpenMule acts as an relay between service providers and end users, which means that users can deploy their CUA agents and traditional workflow AI applications on their own servers, rather than relying on a third-party marketplace. Users can enjoy services on their own computers and control the data they use.
  3. Escrow as a Service: OpenMule handles every transaction, including the payment of fees, the release of escrow, and the release of escrow. This allows users to focus on their business and not worry about the technical details of the transaction, and a better user experience.
  4. Distributed Computing: OpenMule uses distributed computing, which means that it can be used to create a decentralized marketplace. This allows users to create their own marketplace.
  5. Extensible: OpenMule is designed to be easily extended, which means that users can add new features and functionality to the platform as needed, such as physical robots, or physical KVM controllers.
  6. Crypto friendly: OpenMule is designed to be compatible with various cryptocurrencies, including Bitcoin, Ethereum, and others. This allows users to use the platform with their preferred cryptocurrency.
  7. Human in the Loop: OpenMule integrates human in the process, which means that users can interact with the platform and make decisions based on their own needs and preferences, and human workers can compete with AI agents to win the task.
